\CWHeader{Лабораторная работа \textnumero 2}

\CWProblem{
Требуется разработать программу-словарь для хранения пар (слово, число). Программа должна в интерактивном режиме обрабатывать команды добавления, удаления, поиска элементов, а также выполнять сохранение и загрузку всего словаря в компактный бинарный файл.

{\bfseries Вариант структуры данных:} { Реализация словаря на основе структуры данных {PATRICIA}. }

{\bfseries Ограничения на ключи:} { \normalfont\ttfamily Ключи — регистронезависимые слова до 256 знаков латинского алфавита. }
}
