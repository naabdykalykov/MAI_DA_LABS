\CWHeader{Лабораторная работа \textnumero 2}

\CWProblem{
Требуется разработать программу, реализующую один из стандартных алгоритмов поиска образцов в тексте. Программа должна находить все вхождения заданного образца и выводить их местоположение в формате \enquote{номер строки, номер слова}.

{\bfseries Вариант алгоритма:} Поиск одного образца, основанный на построении Z-блоков.

{\bfseries Вариант алфавита:} { \normalfont\ttfamily Слова не более 16 знаков латинского алфавита.}
}
